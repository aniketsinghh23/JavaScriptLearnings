# JavaScript Beginner Basics

This folder is made for complete beginners who want to learn the most important JavaScript fundamentals.

## Topics Included

1. Variables
2. Arrays
3. Objects
4. Functions
5. `for` loop
6. `if` statements

## Folder Structure

- `README.md` - simple explanation of each topic
- `01-variables.js` - variables example
- `02-arrays.js` - arrays example
- `03-objects.js` - objects example
- `04-functions.js` - functions example
- `05-for-loop.js` - for loop example
- `06-if-statements.js` - if statement example

## How to Use

You can open each file one by one and read the comments.
If you have Node.js installed, you can run any file with:

```bash
node 01-variables.js
```

## What You Will Learn

### Variables
Variables store data so you can reuse it later.

### Arrays
Arrays store multiple values in one list.

### Objects
Objects store related data using keys and values.

### Functions
Functions let you reuse code by putting it inside a block.

### `for` Loop
A `for` loop repeats code many times.

### `if` Statement
An `if` statement helps your program make decisions.

## Tip for Beginners

Try changing the values in the examples and run them again. That is one of the fastest ways to learn JavaScript.
